# Tunnel

## Development

It is based on these technologies : 
- Three.js for the 3d rendering
- vite.js for the build 
- TypeScript for type-checking

### `npm i`

Install dependencies

### `npm run dev`

Run the local server

### `npm run build`

Build the project
